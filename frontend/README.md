# Frontend - Shift Board (React)

## Setup
1. In a terminal:
   ```
   cd frontend
   npm install
   npm start
   ```
2. Edit src/config.js to point API_BASE to your backend (e.g. http://localhost:5000/api)

## Notes
- Minimal UI: Login, Dashboard, Shift form (admin), and table to view shifts.
- Uses localStorage for token persistence.
